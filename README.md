# 🛒 ASHADMART - E-Commerce Website

ASHADMART is a modern and responsive E-Commerce website built using HTML, CSS, and JavaScript.  
It features a clean UI with product sections, navigation, categories, and a smooth shopping experience design.

---

## 🚀 Features

- Responsive E-Commerce UI
- Modern Hero Section
- Navigation Bar
- Product Search Bar
- Categories Section
- Shopping Cart Icon
- Smooth and Clean Design
- Mobile Friendly Layout

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
